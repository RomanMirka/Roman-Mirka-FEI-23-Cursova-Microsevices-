Для підключення хедеру потрібно добавити файли header.css/.html/.js у папку з сторінкою. У html коді сторінки де повинен з'явитись хедер добавити *<script src="header.js"></script>*
Якщо виникаютть проблеми:
Опціонально:
*На початок коду (після body): <name>.html добавити: <div id="header-container"></div>
*У файл css: 
#header-container {
    width: 100%;
    position: fixed;
    top: 0;
    left: 0;
    z-index: 1000;
    background-color: #fff;
}
